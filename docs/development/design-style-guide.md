# 課程教學典藏網站 — 改版設計指南  
---

## 1. 配色 (Color Palette)  
| 名稱         | HEX ／ RGB                   | 用途說明                   |
| ---------- | --------------------------- | ---------------------- |
| 主背景色：溫奶杏   | `#F7F1E7` (RGB 247,241,231) | 整體頁面大面積背景，營造溫暖、人文感。    |
| 主要文字色／深木色  | `#4A3F35` (RGB 74,63,53)    | 標題與內文主要文字，確保可讀性。       |
| 標題強調色：金棕   | `#C78E53` (RGB 199,142,83)  | 用於按鈕、重要連結、標籤作點綴。       |
| 中性灰／輔助灰    | `#A8A09B` (RGB 168,160,155) | 用於副文字、說明文字、邊框、背景分隔。    |
| 輔助色：湖水綠    | `#7FB8A4` (RGB 127,184,164) | 標籤、小圖示、分類標籤色使視覺有變化。    |
| 錯誤／警示色：淡珊瑚 | `#E7A69F` (RGB 231,166,159) | 用於提示、錯誤訊息或需要注意的 UI 元件。 |

**使用建議：**  
- 主背景 +深木色文字為主要視覺基礎，營造溫潤且具書卷氣質的基調。  
- 金棕作為主導點綴色，用於「探索課程」「按鈕」等 CTA 元件。  
- 輔助灰與湖水綠為次色，用於標籤、圖示、分類，使版面豐富但不雜亂。  
- 色彩比例建議：背景約 70%，文字約 20%，點綴與輔助色約 10%。  
- 確保文字與背景的對比符合可讀性／可及性（如 WCAG 標準）。

## 2. 字體與排版 (Typography & Hierarchy)  
**字體建議：**  
- 中文正文字體：**Noto Serif TC** 或 **思源宋體** — 書卷感強，適合教育／人文場域。  
- 英文字體：**Montserrat** 或 **Lato** — 清晰現代，與中文正文形成互補。  
- 備選字體（fallback）：中文 serif，英文 sans-serif。

**層級設定：**  
| 層級 | 字體大小 | 行高 | 用途說明 |
|------|----------|-------|-----------|
| H1 | 42px / 2.6 rem | 1.2 | 主頁 Hero 區的大型標題。 |
| H2 | 32px / 2.0 rem | 1.3 | 區塊標題（如「熱門課程」） |
| H3 | 24px / 1.5 rem | 1.4 | 卡片標題、子標題。 |
| H4 | 20px / 1.25 rem | 1.4 | 副標、資訊欄位。 |
| 內文 | 16px / 1.0 rem | 1.6 | 正文敘述、摘要文字。 |
| 小字說明 | 14px / 0.875 rem | 1.4 | 標籤、輔助說明文字。 |

**排版建議：**  
- 文字顏色採深木色 `#4A3F35`，閱讀舒適。  
- 行寬控制在 60–75 字元／行，避免閱讀疲勞。  
- 標題字體加粗（Semi-Bold 或 Bold），段落上下保留足夠空白。  
- 教育網站字體的選擇需兼顧「可讀性」與「溫度」。

## 3. 卡片樣式 (Card Components)  
**基本結構：**  
- 縮圖：卡片頂部放置橫向或正方形照片或插畫（建議比例 4:3 或 1:1）。  
- 課程名稱：使用 H3 樣式（24px），文字為深木色。  
- 年級／學期資訊：14px 小字說明，使用輔助灰 `#A8A09B`。  
- 摘要文字：16px，最多兩行，超過省略為「…」。  
- 標籤列：位於卡片底部，2-3 個圓角標籤（如「黑板畫」「劇本」「歌曲音檔」），背景湖水綠 `#7FB8A4`，文字白色或深木色。  
- 懸停動態：懸停時卡片陰影加深 (例如 box-shadow: 0 4 8 rgba(0,0,0,0.1))，略微放大 (scale 1.02)，標籤背景加深至 `#6BA391`。

**佈局與間距：**  
- 桌面：三／四欄佈局，卡片間距左右 24px，上下 32px。  
- 平板：兩欄佈局。  
- 手機：一欄佈局，左右內距 16px。  
- 卡片內部 padding：圖片下 16px + 文字區 20px + 標籤列 16px。  
- 圓角建議：8px。

**色彩與可讀性：**  
- 卡片背景建議為 白色 `#FFFFFF`，文字為深木色。  
- 標籤字體建議為粗體，標籤背景與文字須為高對比。  
- 縮圖可加奶杏色遮罩 (opacity 0.1–0.2) 以維持色調一致性。

## 4. 按鈕與互動元件 (Buttons & Interactions)  
**按鈕樣式：**  
| 類型 | 樣式說明 | 使用場景 |
|------|----------|-----------|
| 主要 CTA 按鈕 | 背景金棕 `#C78E53`，文字白色 `#FFFFFF`，圓角 6px；hover 背景 `#B07647`，按下背景 `#9B6040`。 | 「探索課程」「加入收藏」等主要操作。 |
| 次要按鈕 | 邊框金棕 2px + 文字金棕，背景透明；hover 背景奶杏 `#F7F1E7`。 | 「了解更多」「回到首頁」等輔助操作。 |
| 標籤／小按鈕 | 背景湖水綠 `#7FB8A4`，文字白色，圓角 4px；hover `#6BA391`。 | 卡片內或分類區的小操作如「查看詳細」「更多標籤」。 |

**動效細節：**  
- hover 狀態：按鈕微放大 (scale 1.02)、陰影 (box-shadow 0 2 4 rgba(0,0,0,0.1))。  
- 點擊回饋：背景色略變、內縮效果。  
- 過渡動畫建議：200-300ms 的 ease-out 模式，流暢自然。

**可用性 & 可及性：**  
- 按鈕文字至少 16px，且高度至少 44px 以增強點擊區域。  
- 背景與文字對比至少 4.5:1。  
- 按鈕支持鍵盤焦點 (focus) 樣式。  
- 可考慮減少動作或提供 “關閉動畫”選項（對 prefers-reduced-motion 支援）。

## 5. 圖像與插畫風格 (Imagery & Illustration)  
- Hero 區建議採用淡雅水彩或手繪插畫，場景可為老師與學生互動、黑板藝術、筆觸感畫面。  
- 課程縮圖建議為：實景照片＋奶杏色遮罩 (opacity 0.1–0.2) 或純插畫風格（如教案書本、劇本道具、音符等），風格一致。  
- 插畫線條建議使用深木色或金棕色細線，並輔以湖水綠點綴。  
- 圖像邊框或卡片建議圓角 8px＋輕微陰影，避免過於平坦。  
- 若文字與圖像重疊，須加遮罩或半透明底色，確保文字可讀性。

---

## 6. 佈局與間距 (Layout & Spacing)  
- 採用 **12欄網格系統**（桌面）作為基礎。  
- 左右內距：桌面約 24px，窄版或內頁可為 16px。  
- 各大區塊上下間距：桌面 96px／平板 60px／手機 40px。  
- 區塊內標題與內容間距：32px。  
- 卡片區左右間距：24px；卡片內 Padding 如前述。  
- 行距與字距統一，內文行距建議為字體大小的 1.4–1.6 倍。

---

## 7. 響應式設計與可及性 (Responsive & Accessibility)  
- 桌面／平板／手機皆適配，卡片排列由三／四欄 →兩欄 →一欄。  
- 所有文字與背景需符合對比率至少 4.5:1。  
- 圖片需有 alt 屬性描述。按鈕需支持鍵盤導航 (focus) 並提供 focus 樣式。  
- 動畫應考慮 prefers-reduced-motion 使用者。  
- 提供語言切換（繁體中文／English），翻譯後仍應符合版面比例與排版風格。

## 8. 使用範例 (Mock Usage)  
- **Hero 區**：背景圖＋奶杏遮罩，H1 標題「課程成為故事，學習化為藝術」，金棕 CTA 按鈕「探索課程」。  
- **卡片區**：白底卡片、圓角、陰影、縮圖＋標題＋年級資訊＋摘要＋標籤。  
- **分類標籤區**：湖水綠圓角按鈕排列、位於卡片區上方。  
- **理念區**：左圖右文排列，背景奶杏 #F7F1E7。  
- **頁腳**：深木色文字、湖水綠連結色、金棕 hover 色；包含聯絡資訊、版權聲明、隱私政策／常見問答連結。

1. **頂部導航 / 標誌區**
    
    - 左上為校或網站標誌。        
    - 右上為語言切換（例如「繁體中文 / English」）。        
    - 導覽列置於頂部，包含 「課程介紹」「所有課程」「留言與交流」等選項。        
    - 導覽列背景採用淺暖色（例如淡奶杏或米膚）與字體為深色，增強對比。
        
2. **主視覺橫幅（Hero Section）**
    
    - 橫跨整頁寬度的大圖或插畫：建議使用溫潤的水彩手繪風格圖像（如教室黑板、水彩筆觸、孩子與老師互動畫面）。        
    - 圖像左或下方覆蓋一段簡短而強烈的標語，例如你提供的文字「This site is a living museum …」可縮為 “課程成為故事，學習化為藝術”＋“Browse the journeys co-created by teachers and students”。        
    - 橫幅下方放置一顆「探索課程」主要 CTA 按鈕。按鈕建議採暖色調，並在滑鼠懸停時稍作變化（如深色邊框或陰影浮動）。
        
3. **課程卡片區（推薦課程集錦）**
    
    - 接下來一個區塊為「熱門／最新課程」展示。採三欄或四欄響應式格子佈局（桌面端三欄，平板或手機端自動調整為兩列或一列）。        
    - 每張卡片含：縮圖（可為課程實景照片或藝術照）、課程名稱（如「能高山 班級戲劇」）＋年級／學期資訊、小段課程摘要。        
    - 卡片底部添入標籤（如「黑板畫」「照片集」「劇本」）以增加內容識別。卡片本體使用輕微陰影或懸停時浮動效果，提升互動感。
        
4. **課程分類 & 快速導航**
    
    - 在卡片區下方，可再設一列薄橫條或背景塊，作為「課程分類」入口，如「年級」「自然主題」「藝術課程」「語文主課」「歷史主課」。點選分類可快速篩選內容。        
    - 每個分類圖示或標籤使用一致圓角矩形按鈕、配以小圖標（例如書本、畫筆、地球儀）以輔助視覺辨識。
        
5. **網站理念區 + 社群／教師推薦**
    
    - 再往下可加入一區「我們的理念」簡短段落，簡述：「典藏不是冷冰冰的資料庫，而是教師與學生共創的生命歷程。」搭配一張教師與學生互動情境照片或插畫。        
    - 接著「教師／學生推薦語」滑塊（carousel）顯示 2-3 則語錄：例如「這門課讓我重新看見學習的美好」等。        
    - 最底部為頁腳：包含聯絡資訊（如 it@waldorf.ilc.edu.tw）、版權聲明（如© 2025 慈心華德福），連結欄（隱私政策、網站地圖）等。

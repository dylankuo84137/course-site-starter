Guide for AI Tools

This page explains how AI tools can effectively parse and utilize the educational content on this site.

Site Architecture Overview

Cixin Course Weaver (æ…ˆå¿ƒèª²ç¨‹ç¹”åœ–) is a static educational website built with Eleventy (11ty), Waldorf's bilingual course archive, connecting lesson goals with teaching artifacts, media folders, and downloads for planners and partners.

Main Page Structure

  • Home (/): Site introduction, search portal, and course highlights
  • Language Pages:
    • /zh-TW/ - Traditional Chinese homepage
    • /en-US/ - English homepage
  • All Courses (/courses/): Complete course catalog with filtering by grade, domain, and teacher
  • Course Pages (/courses/{slug}/): Each course includes:
    • Main Pages:
      • /courses/{slug}/ - Course homepage (overview, learning objectives, teacher bio, downloadable materials)
      • /courses/{slug}/workbook/ - Student workbook photos with tag filtering
      • /courses/{slug}/photos/ - Classroom photo gallery with tag filtering
      • /courses/{slug}/blackboard/ - Blackboard drawing artwork with tag filtering
      • /courses/{slug}/scripts/ - Play script performance photos with tag filtering
      • /courses/{slug}/songs/ - Songs and audio recordings with inline playback
      • /courses/{slug}/videos/ - Video performances (YouTube embeds)
      • /courses/{slug}/sheet-music/ - Sheet music PDFs with preview
      • /courses/{slug}/syllabus/ - Course syllabus PDFs with preview
      • /courses/{slug}/worksheet/ - Student worksheet PDFs with preview
    • Google Docs Content Pages:
      • /courses/{slug}/description/ - Course description from Google Doc
      • /courses/{slug}/story/ - Story content from Google Doc
      • /courses/{slug}/play-script/ - Play script from Google Doc
  • Reflections (/reflections/): Community discussion forum
  • AI-Specific Pages:
    • /for-ai/ - This guide for AI tools
    • /ai-index.json - Complete course catalog with full text content
    • /feed.json - Course update timeline (JSON Feed format)
    • /sitemap.xml - Complete site sitemap

What's Included in ai-index.json

The AI index now contains complete Drive-synced content:

  • Google Docs full text (docs.course_description.content, docs.play_script.content): Plain text extracted from Google Docs, ideal for semantic search and RAG (Retrieval-Augmented Generation)
  • Image metadata (driveContent.workbook_photos, driveContent.blackboard, driveContent.photos): File IDs, names, tags, and multiple URL formats (thumbnail, preview, direct view)
  • Audio file metadata (driveContent.songs_audio): Audio file names, tags, MIME types, and stream/download URLs
  • PDF documents (driveContent.sheet_music, driveContent.play_scripts): Sheet music and script PDFs with viewer URLs

Content Types & Processing Recommendations

Text Content

  • Course descriptions: Teaching philosophy, thematic explanations, learning objectives
  • Teacher bios: Background, teaching style
  • Google Docs integration: Some courses include full document content

Visual Materials

  • Workbooks: Student learning process records
  • Blackboard drawings: Teacher chalkboard art
  • Photo galleries: Classroom activities and events
  • Script photos: Drama performance documentation

Audio Content

  • Formats: Primarily M4A, MP3
  • Types: Course songs, recitations, instrument performances
  • Access: Stream or download via Google Drive API

PDF Documents

Many course materials are in PDF format, including:

  • Worksheets (driveContent.worksheet): Student exercise templates, note-taking formats
  • Sheet Music (driveContent.sheet_music): Musical scores and notation
  • Play Scripts (driveContent.play_scripts): Drama performance scripts
  • Syllabi (driveContent.syllabus): Course outlines and learning plans

AI Usage Guidelines

Recommended Tools & Methods

NotebookLM

  1. Add source: Select "Website"
  2. Paste course URL (e.g., https://dylankuo84137.github.io/course-site-starter/courses/2a-nenggao-113-summer/)
  3. NotebookLM will crawl the page and linked content
  4. Example query: "What are the learning objectives of this course?"

ChatGPT / Claude

  1. Download https://dylankuo84137.github.io/course-site-starter/ai-index.json
  2. Upload the JSON file to the AI chat
  3. Ask questions like: "Analyze the teaching philosophy in the Mt. Nenggao course based on the teacher's description"
  4. The AI can access full Google Docs text, image lists, and audio file metadata

Sample Queries

  • "Analyze the teaching strategies in Grade 2 nature main lesson"
  • "Compare the evolution of blackboard drawing styles across grade levels"
  • "How does the drama course integrate Indigenous Taiwanese culture?"
  • "What learning processes are reflected in the workbooks?"
  • "List all songs used in the Mt. Nenggao drama course"
  • "Extract keywords from the play script and course description"

Multi-Language Support

All course content is available in both Traditional Chinese (zh-TW) and English (en-US) via the i18n object. AI tools can:

  • Access Chinese content: course.i18n["zh-TW"].title
  • Access English content: course.i18n["en-US"].title
  • Use metadata for filtering: course.metadata.grade_level, course.metadata.domain_category
